== Installation ==

1. Upload `heymarket-widget.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create and/or login to your [Heymarket Account](https://app.heymarket.com/?ref=wp-plugin)
4. [Configure the Heymarket Widget](https://app.heymarket.com/team/integrations/widget/?ref=wp-plugin) under Manage Integrations in your Heymarket account to get a Client ID.
3. In the admin dashboard of your Wordpress site, open the Heymarket settings page and input the Client ID from your Heymarket Widget setup.
4. Any changes made to the Widget in your Heymarket account will now show up automatically on your Wordpress site.
